---
title: "EZAnim:stopAll() — Stop All of a Model's Animations"
sidebarTitle: "stopAll()"
description: "Stop every active animation on a model at once and fully reset all EZAnim tracking state, including per-group and cached track records, for that model."
---

`:stopAll()` is a hard reset for a model's entire animation state. It iterates every track EZAnim has loaded for the model, stops any that are still playing, and then clears all associated tracking state — cached tracks, the current-animation record, paused-track records, and per-group current-track records. Use it in situations where you need a guaranteed clean slate: character death, respawn sequences, cutscene transitions, or any moment where the previous animation state is no longer meaningful.

```lua EZAnim:stopAll() signature
EZAnim:stopAll(Model: Model, FadeTime: number?)
```

## Parameters

<ParamField query="Model" type="Model" required>
  The `Model` instance whose animations should all be stopped and whose internal EZAnim state should be cleared.
</ParamField>

<ParamField query="FadeTime" type="number">
  The fade-out duration in seconds applied to every currently playing track. Overrides `DefaultFade` for this call only. Pass `0` to cut all animations instantly.
</ParamField>

## What gets cleared

After `:stopAll()` returns, all EZAnim state for the given model is fully reset:

| State | Purpose | Cleared |
|---|---|---|
| Track cache | Cached `AnimationTrack` objects for each animation ID | ✓ |
| Current-animation record | Which animation is currently active on the model | ✓ |
| Paused-track record | Any tracks that were paused rather than stopped | ✓ |
| Per-group current record | Which animation is currently active per logical group | ✓ |

Because the track cache is wiped, the next call to `:play()` on this model will reload and re-cache tracks from scratch, exactly as if the model had just been initialized.

## Example

```lua Script — handle character death with a clean animation reset
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local Anim = require(ReplicatedStorage.AnimModule)

local character = script.Parent
local humanoid  = character:WaitForChild("Humanoid")

humanoid.Died:Connect(function()
    -- Immediately wipe every active animation with a short fade
    Anim:stopAll(character, 0.1)

    -- Play the death animation from a clean state
    Anim:playOnce(character, "Death", "Action4", false, 0.1)
end)

-- Respawn cleanup: when the character is about to be removed,
-- ensure no dangling tracks remain even if AutoCleanUp is disabled
character.AncestryChanged:Connect(function(_, parent)
    if parent == nil then
        Anim:stopAll(character, 0)
    end
end)
```
