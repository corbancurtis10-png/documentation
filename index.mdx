EZAnim is a lightweight Luau module for Roblox that handles all the boilerplate of animation management — loading tracks, controlling playback, queuing sequences, and cleaning up after models. Instead of wiring up Animators and Animation instances by hand, you describe your animations once and let EZAnim do the rest.

## Why EZAnim?

Managing Roblox animations manually means creating `Animation` instances, finding `Humanoid` or `AnimationController` objects, loading tracks, and remembering to stop everything when a model is destroyed. EZAnim collapses that entire workflow into a single module with a clean API.

## Quick Example

```lua EZAnim example
local EZAnim = require(game.ReplicatedStorage.EZAnim)

local Anim = EZAnim.init({
    List = {
        Idle   = "rbxassetid://123456789",
        Run    = "rbxassetid://987654321",
        Jump   = "rbxassetid://111222333",
    },
    DefaultFade  = 0.15,
    DefaultSpeed = 1,
})

-- Play the Idle animation on a character model
Anim:play(character, "Idle", "Idle", true)
```

<Steps>
  <Step title="Install EZAnim">
    Grab EZAnim from the [Roblox Creator Store](https://create.roblox.com/store/asset/133114641753550/EZAnim) and place it in `ReplicatedStorage`.
  </Step>
  <Step title="Initialize">
    Call `EZAnim.init()` with your animation ID list and default settings to create a controller instance.
  </Step>
  <Step title="Play Animations">
    Use `Anim:play()` to start any animation on any model. EZAnim handles track loading and caching automatically.
  </Step>
  <Step title="Control & Clean Up">
    Pause, resume, queue, cross-fade, and stop animations. EZAnim cleans up automatically when models are destroyed.
  </Step>
</Steps>
